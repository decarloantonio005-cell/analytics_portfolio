
***Project Overview***

This project analyzes a transactional sales dataset to extract business insights related to revenue trends, product performance, and customer demographics.
The analysis was conducted using SQL for data exploration and aggregation, while Power BI was used to build an interactive dashboard for data visualization and storytelling.

The objective is to demonstrate end-to-end analytical skills, from raw data querying to business-oriented insights and visualization.

***Dataset Description***

Granularity: One row per transaction

Main fields:
Date
Product Category
Quantity
Price per Unit
Total Amount
Gender
Age
